<pre><code>[ pcap-analyzer ] (C Parser)

       ▼ (Trigger)
[ incident-tracker-api ] (Flask REST RBAC) ◄─── [ llm-triage ] (AI Guardrails)

       ▼
[ cyber-dashboard ] (MTTD / MTTR Real-time metrics)

[ dora-compliance-engine ] ──────────────────────► CI/CD Gate (DORA / ISO 27001)
</code></pre>
